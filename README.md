# 🏥 Salutis da Amazônia

Site institucional e catálogo digital da **Salutis da Amazônia**, empresa especializada no fornecimento de materiais hospitalares e participação em processos licitatórios.

O projeto foi desenvolvido para apresentar a empresa, seus produtos e facilitar o contato com clientes através do WhatsApp.

---

## 📌 Sobre o Projeto

O site apresenta:

- História da empresa
- Catálogo de produtos hospitalares
- Carrinho de pedidos
- Seleção de produtos com variações
- Envio do orçamento diretamente pelo WhatsApp
- Localização da empresa via Google Maps
- Links para redes sociais

---

## 🏥 Produtos

Atualmente o catálogo possui:

- Luvas hospitalares
- Máscaras
- Aventais
- Equipos
- Seringas
- Agulhas

Novos produtos poderão ser adicionados futuramente.

---

## 🚀 Funcionalidades

✅ Layout responsivo para computador e celular

✅ Menu fixo de navegação

✅ Catálogo de produtos

✅ Carrinho em formato de pop-up

✅ Adicionar e remover produtos

✅ Seleção de tamanho e cor

✅ Finalização do pedido pelo WhatsApp

✅ Integração com localização
